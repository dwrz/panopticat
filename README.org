* Raspberry Pi Setup
1. Download Raspbian image.
2. Copy image to MicroSD card.
3. Enable SSH and add WiFi credentials.
4. Boot and SSH into RPi.
5. Change default password.
6. Update system.
7. Run raspi-config.
8. Setup new user.
9. Setup SSH keys, firewall (ufw), fail2ban.
10. Install Docker.
